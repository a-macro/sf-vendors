Выберите тип вопроса для добавления

  • Позволяет выбирать один или несколько правильных ответов из заданного списка.
  • Простая форма вопроса «Множественный выбор», предполагающая только два варианта ответа: «Верно» или «Неверно».
  • Ответ на каждый из нескольких вопросов должен быть выбран из списка возможных.
  • Позволяет вводить в качестве ответа одно или несколько слов. Ответы оцениваются путем сравнения с разными образцами ответов, в которых могут использоваться подстановочные знаки.
  • Позволяет сравнивать числовые ответы с несколькими заданными вариантами с учетом единиц измерения. Возможен и учет допустимых погрешностей.
  • Допускает в ответе загрузить файл и/или ввести текст. Ответ должен быть оценен преподавателем вручную.
  • Вопросы такого типа являются очень гибкими, но могут быть созданы только путем ввода текста со специальными кодами, которые создают встроенные вопросы «Множественный выбор», «Числовой ответ» и «Короткий ответ».
  • Пропущенные слова в тексте вопроса заполняются с помощью выпадающих меню.
  • Вычисляемые вопросы подобны числовым вопросам, только в них используются числа, которые случайно выбираются из набора при прохождении теста.
  • Множественные Вычисляемые вопросы устроены так же, как вопросы типа «Множественный выбор», с тем отличием, что ответами в них служат числовые результаты формул. Значения в формулах выбираются из заранее определенного набора значений случайным образом при прохождении теста.
  • Пропущенные слова в тексте заполняются с помощью перетаскивания
  • Маркеры перетаскиваются на фоновое изображение.

    Примечание. Этот тип вопросов недоступен для пользователей с плохим зрением.
  • Изображение или текст необходимо перетащить в зону на фоновом изображении.

    Примечание. Этот тип вопросов недоступен для пользователей с ослабленным зрением.
  • Более простая версия вычисляемых вопросов, которые подобны числовым вопросам, но с использованием чисел, выбираемых случайным образом из определенного набора при прохождении теста.
